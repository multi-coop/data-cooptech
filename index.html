Intégration du widget gitribute-file dans une page html

L'import du widget gitribute dans une page html permet d'afficher sur un site tiers

L'import du widget consiste à simplement copier-coller quelques lignes dans la page html de votre choix.

Le widget est paramétrable et il peut être utilisé indistinctement et séparément pour différents tableurs source

Le widget est sous licence libre et il n'y a aucun coût ou surcoût dans son utilisation.

Plusieurs fonctionnalités sont intégrées :

  • Filtres de sélection (paramétrables) ;
  • Redirection vers le site source depuis les fiches détaillées ;
  • Rechargement des données source ;
  • Possibilité de télécharger les données au format .tsv ;
  • Plus de d'infos dans la documentation en ligne.
